> 作为一个垃圾佬，我实在不想购置一个好的硬盘安装，所以这使我不得不重装系统。
> 因此，我决定创建一个项目，将我日常在NAS上使用的命令集结成一个便捷的小工具箱。这样一来，我就可以减少敲点命令😭😭😭
